연운(Where Winds Meet) 게임 체크리스트

본 프로젝트는
연운 글로벌 서버 ‘출입금지’ 동맹을 기준으로 제작된
일일 / 주간 / 월간 루틴 관리용 체크리스트입니다.

제작 목적

동맹원들이 놓치기 쉬운 일일·주간·월간 콘텐츠를 한 눈에 정리

개인 진행 상황을 간단히 체크할 수 있도록 지원

디스코드 공지나 구두 전달을 보완하기 위한 참고용 도구

사용 방식

체크 상태는 각자 사용하는 브라우저의 로컬 저장소(localStorage)에만 저장됩니다.

다른 사람의 체크 상태와 공유되거나 섞이지 않습니다.

로그인, 계정 연동, 서버 통신은 사용하지 않습니다.

개인 체크용으로 부담 없이 사용하시면 됩니다.

대상
연운 게임을 플레이하는 유저

‘출입금지’ 동맹 기준으로 구성되어 있으나
필요하시면 링크를 공유하여 사용하셔도 괜찮습니다.

공유에 대해

본 체크리스트 페이지는 동맹원 간 자유롭게 공유 가능합니다.

외부 공유 시에도 개인 정보는 수집되지 않으며
각 사용자 환경에만 체크 정보가 저장됩니다.

문의 및 수정 제안

기능 추가, 항목 수정 등 문의 사항은
출입금지 동맹 디스코드의 문의/건의 게시판을 통해 전달 부탁드립니다.
